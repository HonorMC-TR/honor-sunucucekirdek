# HonorMC Yol Haritasi

## Faz 0 - Cekirdek Tabani

- Purpur `26.2` fork tabani kurulur.
- Patch sistemi dogrulanir.
- HonorMC jar paketleme gorevi eklenir.
- GitHub Actions release artifact uretir.
- Upstream guncelleme kontrolu otomatiklesir.

## Faz 1 - HonorMC Kimligi

- Konsol banneri HonorMC olarak degisir.
- Manifest marka alanlari HonorMC'ya tasinir.
- README, dokuman ve kurulum dili Turkce olur.
- Lisans ve upstream atiflari korunur.

## Faz 2 - Temiz Konsol

- Log profilleri cekirdege eklenir.
- Plugin acilis/kapanis satirlari secilebilir hale gelir.
- Chunk, watchdog, debug ve noisy logger filtreleri config'e baglanir.

## Faz 3 - Performans Profilleri

- SMP, Survival, Skyblock, BoxPvP, Faction ve Minigame profilleri gelir.
- Her profil Paper/Purpur configleri icin guvenli varsayilanlar uretir.
- Agresif optimizasyonlar eklenti uyumlulugu riskiyle birlikte etiketlenir.

## Faz 4 - Coklu Surum Stratejisi

- Stabil release sonrasinda protokol uyumlulugu arastirilir.
- Gerekirse ViaVersion benzeri harici eklentiye mecbur kalmadan temel aralik destegi icin ayri katman tasarlanir.
- Cekirdek dallari `honormc/<minecraft-version>` duzenine oturtulur.

## Faz 5 - Launcher ve Sunucu Uygulamasi

- HonorMC Baslatici Java kontrolu, RAM ayari, profil secimi ve jar guncelleme kontrolu yapar.
- Daha sonra masaustu sunucu yonetim uygulamasi gelistirilebilir.
